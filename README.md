## เกี่ยวกับหลักสูตร

หลักสูตร Machine Learning สำหรับผู้เริ่มต้น ครอบคลุมตั้งแต่พื้นฐาน AI ไปจนถึงการสร้างโมเดล Classification, Regression, Clustering ด้วย Python และ scikit-learn พร้อม Workshop ลงมือทำจริงกับข้อมูลคุณภาพน้ำ

**ระยะเวลา:** 2 ชั่วโมง | **ระดับ:** เริ่มต้น — กลาง (Beginner — Intermediate)

## สิ่งที่จะได้รับ

- อธิบายความแตกต่างระหว่าง AI, Machine Learning และ Deep Learning ได้
- สร้างโมเดล Classification ด้วย Decision Tree และ KNN
- สร้างโมเดล Regression ด้วย Linear Regression
- จัดกลุ่มข้อมูลด้วย K-Means Clustering
- เข้าใจ ML Pipeline ตั้งแต่เตรียมข้อมูลจนถึงประเมินผล
- วิเคราะห์ข้อมูลจริงและสร้าง AI คัดกรองคุณภาพน้ำ
